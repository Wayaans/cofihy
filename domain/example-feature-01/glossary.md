# Glossary: Example Feature

> Domain terms and definitions for the Example Feature.

| Term       | Definition                                                        |
| ---------- | ----------------------------------------------------------------- |
| Item       | A generic resource entity managed through CRUD operations         |
| Owner      | The user who created an item and has full control over it         |
| Status     | The current state of an item: `active` or `archived`             |
| Archive    | Soft removal -- item becomes hidden from default lists            |
| Restore    | Returning an archived item back to `active` status               |
